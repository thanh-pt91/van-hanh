🔑 Google OAuth (Đăng nhập bằng Google)
Cho phép người dùng đăng nhập bằng tài khoản Google thực sự thay vì nhập email thủ công.
  1. Vào console.cloud.google.com → Tạo project mới
  2. APIs & Services → Credentials → Create OAuth 2.0 Client ID
  3. Application type: Web application
  4. Authorized JavaScript origins: https://thanh-pt91.github.io
  5. Copy Client ID → dán vào ô trên → Lưu
⚡ Lỗi "You do not have permission to call DriveApp"?
→ Bạn chưa cấp quyền Drive cho script. Làm bước 3 bên dưới trước khi Deploy!
📋 Hướng dẫn cài đặt (làm đúng thứ tự):
  1. Nhấn 🎲 Tạo ngẫu nhiên để tạo Secret Token → 📋 Copy
  2. Mở Google Sheets → Extensions → Apps Script
  3. Xóa code mặc định → nhấn Copy bên dưới → dán vào → Save (Ctrl+S)
  4. 🔐 Tìm dòng const SECRET_TOKEN = 'THAY_BANG_TOKEN_CUA_BAN' → thay bằng token vừa copy
  5. ⚡ Chọn hàm setup → nhấn Run ▶ → Allow
  6. Click Deploy → New Deployment → Web App | Execute as: Me | Access: Anyone
  7. Copy URL → dán vào ô URL trên → nhấn 💾 Lưu & Kết nối
